## Run Locally

Clone the project

```bash
  git clone https://github.com/stupid-kid-af/Gradient-Generator-Javascript.git
```

Go to the project directory

```bash
  cd Gradient-Generator-Javascript
```

Start the server

```bash
  google-chrome index.html
            or
  <Browser> index.html
```
